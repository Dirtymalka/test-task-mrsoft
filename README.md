# Тестовое задание #

## Выполненные требования:

 - [x] При вводе в поле числа и клике по кнопке фильтра по длине слов в место для вывода данных выводятся строки с длиной строки больше введенного числа.
 - [x] При вводе строки в поле и клике по кнопке фильтра по подстроке в место для вывода данных выводятся строки, которые содержат введенную подстроку.
 - [x] Если checkbox чувствительности регистра проставлен, строки сравниваются с учетом регистра ('aAa' не равно 'aaa'). Если checkbox снят, строки сравниваются без учета регистра ('aAa' равно 'aaa').
